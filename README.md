# Hotel Booking System

A hotel booking web application I built as a school project.

## What it does

- Users can register and login
- Browse hotels and rooms
- Search for available rooms by date
- Make and cancel bookings
- Admin can add/edit/delete hotels and rooms

## Technologies used

- Java Spring Boot
- MySQL database
- HTML, CSS, JavaScript
- Maven for dependencies

## How to run it

1. Make sure MySQL is running
2. Create database called hotel_db
3. Open terminal in project folder
4. Run: ./mvnw spring-boot:run
5. Open browser to: http://localhost:8080

## Live Demo

Run locally: http://localhost:8080
- Login page: http://localhost:8080/login.html
- Register page: http://localhost:8080/register.html
